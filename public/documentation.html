Simple css class names for inputs

.field Always put inputs inside it
this is like the .form-control in bootstrap or the .field in bulma css. You can put in all kind of inputs
Example with text and number inputs.
.input-radio
.input-range
.input-checkbox
For inputs of type radio, range and checkbox.
With this types of inputs you have to put in inside a label tag with the correspond class name.
Example with inputs of type radio, range and checkbox.
textarea and select
This tags are very simple, just put in inside a .field container.
